## Install

[![Greenkeeper badge](https://badges.greenkeeper.io/tiaanduplessis/timed-cmd.svg)](https://greenkeeper.io/)

```sh
$ npm install --global timed-cmd
# OR
$ yarn global add timed-cmd
```

## Usage

```sh
$ timed-cmd npm i
#
#⌨ Command: npm i
#⌛ Time: 17.408053188 s
```
